# Сниппеты и шаблоны

Эта папка предназначена для хранения шаблонов функций, компонентов и комментариев для Cursor AI.

## Использование

Здесь можно хранить:
- Шаблоны компонентов
- Шаблоны функций
- Шаблоны комментариев
- Другие переиспользуемые паттерны кода

## Примеры

### Шаблон компонента
```
components/
  ComponentName/
    ComponentName.js
    ComponentName.scss
```

### Шаблон функции
```javascript
export function functionName(params) {
  // Описание функции
  // @param {type} params - описание параметров
  // @returns {type} - описание возвращаемого значения
}
```
